01
⚡
AGENT MODE

Switch to Agent Mode — stop using plain chat

Agent mode edits files, runs commands, installs packages, and fixes errors autonomously in a loop.

  • Copilot Chat → switch to Agent
  • Describe what, not how
  • Creates files, runs tests, fixes failures
"Chat answers questions. Agent ships features."
// One prompt in Agent mode:
> "Add /health endpoint with tests"

// Agent executes:
✓ Created src/routes/health.ts
✓ Created tests/health.test.ts
✓ Updated README.md
✓ Ran npm test — 3 passed

  • Try Claude / Codex agents too
  • Signs commits & runs CI
