01
⚡
IDE · AGENT MODE

Switch to Agent Mode — stop using plain chat

Agent mode edits files, runs terminal commands, installs packages, and fixes errors in a loop — autonomously.

  • Open Copilot Chat → switch dropdown to Agent
  • Describe what you want, not how to build it
  • Copilot creates files, runs tests, fixes failures
"Chat answers questions. Agent ships features."
// You type one prompt:
> "Add a /health endpoint with tests
  and update the README"

// Agent executes:
✓ Created src/routes/health.ts
✓ Created tests/health.test.ts
✓ Updated README.md
✓ Ran npm test — 3 passed
02
🤖
CLOUD AGENT

Assign issues directly to Copilot

Cloud Agent runs on GitHub — no IDE needed. Assign an issue and it plans, codes, and opens a PR.

  • Issue → Assignees → Copilot
  • Also try Claude or Codex agents
  • Agent signs its commits & runs CI
"Your backlog has an AI teammate."
03
📋
CONFIG

Add custom instructions — once

Create .github/copilot-instructions.md. Your rules apply every session, automatically.

  • Define language & framework conventions
  • Set testing & style rules
  • Org-level instructions now GA
"Write rules once. Applied forever."
04
🧠
MODEL SELECTION

Pick the right model for the task

Claude Opus 4.7 for deep refactors. GPT-5.4 mini for fast, unlimited chats. Match model to complexity.

  • Opus 4.7 — complex multi-file work
  • GPT-5.4 mini — quick unlimited chats
  • Gemini / Codex — alternative strengths
"Best model ≠ biggest model."
05
📁
PROMPT FILES

Create reusable prompt files

Store prompts in .github/prompts/ and invoke with / slash command. Share across your team.

.github/prompts/code-review.md
.github/prompts/add-tests.md
"One great prompt > 50 random asks."

🔌
MCP SERVERS

Connect external tools via MCP

Databases, Docker, Azure, 50+ integrations. Copilot reads your full tool ecosystem mid-conversation.

  • Configure in .vscode/mcp.json
  • Browse github.com/mcp registry
  • Org admins manage MCP allow-lists
"Copilot becomes infrastructure."
07
🖥️
CLI

Copilot CLI — natural language terminal

gh copilot turns plain English into shell commands. Now supports BYOK and local models.

  • Ask in English, get the exact command
  • BYOK + local models (April 2026)
  • Works on GitHub Mobile too
"Stop Googling terminal commands."
08

IDE SHORTCUTS

Master @workspace, @terminal, and slash commands

Give Copilot full repo context with @workspace. Use @terminal for CLI help. Fire off /fix, /explain, /tests, /doc for instant actions without typing paragraphs.

@workspace "how does auth work?" /fix → auto-fixes selection /tests → generates tests
"Shortcuts beat paragraphs. Every time."
