# Social Post Drafter

Draft LinkedIn post copy for carousels and technical content with Unicode formatting.

## Unicode Formatting

Use Mathematical Sans-Serif characters for formatting:
- **Bold**: 𝗔-𝗭, 𝗮-𝘇, 𝟬-𝟵
- **Italic**: 𝘈-𝘡, 𝘢-𝘻
- **Bold-Italic**: 𝘼-𝙕, 𝙖-𝙯

Reference `~/.copilot/skills/linkedin-post-formatter/references/unicode-charmap.md` for full mapping.

## Carousel Post Pattern

```

Context — why this matters (2-3 lines)

𝟭. Tip title (bold number)
𝟮. Tip title

𝟭𝟮. Tip title
```
