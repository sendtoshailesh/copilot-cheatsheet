## 6. Key Themes Extracted

### Theme 1: Design Thinking × SDLC Integration
- Workshop structure mapped to full software development lifecycle
- AI-DLC (AI Development Lifecycle) as evolution of DT
- Planning → Design → Development → Testing → Deployment → Operations

### Theme 2: HVE (Hypervelocity Engineering) Foundation
- Design Thinking is a core pillar of HVE methodology
- HVE-Core GitHub repo contains DT agents (`dt-coach`, `dt-tutor`)
- Official Viva Learning training path available

### Theme 3: Customer-Facing Workshop Delivery
- Active workshops planned for India and Asia-region customers
- Sagility identified as specific customer opportunity
- Chief Advisor positioning for Asia team
- Facilitation shadowing and content sharing model

### Theme 4: Three DT Spaces
- **Problem Space** — Research, discovery, empathy
- **Solution Space** — Ideation, prototyping, validation
- **Implementation Space** — Delivery, measurement, iteration

### Theme 5: Internal Practice-Building
- Institutionalizing DT across Dev Advisory team
- Creating canonical documentation and asset maps
- Building regional champion network
